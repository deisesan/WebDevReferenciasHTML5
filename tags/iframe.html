<iframe>

A tag <iframe> específica um quadro embutido.

Um quadro embutido é usado para incorporar outro documento no documento HTML atual.

Dica: Use CSS para estilizar <iframe>.

Dica: É uma boa prática sempre incluir um atributo title para o arquivo <iframe>. Isso é usado por leitores de tela para ler qual é o conteúdo do arquivo <iframe>.

Exemplo

Um quadro embutido é marcado da seguinte forma:

<iframe src="https://www.w3schools.com" title="W3Schools Free Online Web Tutorials"> </iframe>

Suporte ao navegador

A tag <iframe> é compatível com todos os principais navegadores.

Atributos

Atributo Valor Descrição
allow - Especifica uma política de recursos para o <iframe>
allowfullscreen true
false		 Defina como true se <iframe> puder ativar o modo de tela inteira chamando o método
allowpaymentrequest true
false		 Definido como verdadeiro se um <iframe> de origem cruzada tiver permissão para invocar a API de solicitação de pagamento
height pixels Especifica a altura de um <iframe>. A altura padrão é 150 pixels
loading eager
lazy		 Especifica se um navegador deve carregar um iframe imediatamente ou adiar o carregamento de iframes até que algumas condições sejam atendidas
name text Especifica o nome de um <iframe>
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
same-origin
strict-origin-when-cross-origin
unsafe-url		 Especifica quais informações de referência enviar ao buscar o iframe
sandbox allow-forms
allow-pointer-lock
allow-popups
allow-same-origin
allow-scripts
allow-top-navigation		 Habilita um conjunto extra de restrições para o conteúdo em um <iframe>
src URL Especifica o endereço do documento a ser incorporado no <iframe>
srcdoc HTML_code Especifica o conteúdo HTML da página a ser exibida no <iframe>
width pixels Especifica a largura de um <iframe>. A largura padrão é 300 pixels

Mais exemplos

Adicionar e remover bordas de iframe (com CSS):

<iframe src="/default.asp" width="100%" height="300" style="border:1px solid black;">
</iframe>

<iframe src="/default.asp" width="100%" height="300" style="border:none;">
</iframe>

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <iframe> com os seguintes valores padrão:

iframe:focus {
   outline: none;
}

iframe[seamless] {
   display: block;
}
