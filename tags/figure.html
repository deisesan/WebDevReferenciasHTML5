<figure>

A tag <figure> específica conteúdo independente, como ilustrações, diagramas, fotos, listagens de código etc.

Embora o conteúdo do elemento <figure> esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deve afetar o fluxo do documento.

Dica: O elemento <figcaption> é usado para adicionar uma legenda ao elemento <figure>.

Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento <figcaption> para definir uma legenda para a foto:

<figure>
  <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
  <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Suporte ao navegador

A tag <figure> é compatível com todos os principais navegadores, mas ela é nova no HTML5, portanto, você deve usar em versões mais recentes dos navegadores.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <figure> com os seguintes valores padrão:

figure {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 40px;
  margin-right: 40px;
}
