<param>

A tag <param> é usada para definir parâmetros para um elemento <object>.

Exemplo

Defina o parâmetro "autoplay" como "true", para que o som comece a tocar assim que a página for carregada:

<object data="horse.wav" >
  <param name="autoplay" value="true">
</object>

Suporte ao navegador

A tag <param> é compatível com todos os principais navegadores. No entanto, o formato de arquivo definido em <object> pode não ser suportado em todos os navegadores.

Atributos

Atributo Valor Descrição
name name Específica o nome de um parâmetro
value value Específica o valor do parâmetro

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <param> com os seguintes valores padrão:

param {

   display: none;

}
