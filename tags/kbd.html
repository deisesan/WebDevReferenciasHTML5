<kbd>

A tag <kbd> é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Dica: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS

Exemplo

Define algum texto como entrada de teclado em um documento:

<p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p>

<p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>

Suporte ao navegador

A tag <kbd> é compatível com todos os principais navegadores.

Exemplo

Use CSS para definir o estilo do elemento <kbd> :

<html>
<head>
<style>
kbd {
  border-radius: 2px;
  padding: 2px;
  border: 1px solid black;
}
</style>
</head>
<body>

<p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p>
<p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>

</body>
</html>

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <kbd> com os seguintes valores padrão:

kbd {
  font-family: monospace;

}
