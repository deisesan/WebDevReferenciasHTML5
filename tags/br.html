<br>

A tag <br> insere uma única quebra de linha.

A tag <br> é útil para escrever endereços ou poemas.

A tag <br> é uma tag vazia, o que significa que não tem tag final.

Exemplo

Inserir quebras de linha simples em um texto:

<p>To force<br> line breaks<br> in a text,<br> use the br<br> element.</p>

Resultado:

To force
line breaks
in a text,
use the br
element.

Dicas e Notas

Use a tag <br> para inserir quebras de linha, não para adicionar espaço entre parágrafos.

Suporte ao navegador

A tag <br> é compatível com todos os principais navegadores.

Mais exemplos

Use <br> em um poema:

<p>Be not afraid of greatness.<br>
Some are born great,<br>
some achieve greatness,<br>
and others have greatness thrust upon them.</p>

<p><em>-William Shakespeare</em></p>

Resultado:

Be not afraid of greatness.
Some are born great,
some achieve greatness,
and others have greatness thrust upon them.

-William Shakespeare
