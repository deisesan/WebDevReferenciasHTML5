<figcaption>

A tag <figcaption> em HTML é usada para definir uma legenda para o elemento <figure> em um documento. O elemento <figcaption> deve, obrigatoriamente, possuir um elemento <figure> como pai.

Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento <figcaption> para definir uma legenda para a foto:

<figure>
  <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
  <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Suporte ao navegador

A tag <figcaption> é compatível com todos os principais navegadores, mas ela é nova no HTML5, portanto, você deve usar em versões mais recentes dos navegadores.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <figcaption> com os seguintes valores padrão:

figcaption {

 display: block;

}
